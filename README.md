# EcoLazo

Proyecto Java de EcoLazo.

## Como colaborar

1. Clona el repositorio:

   ```bash
   git clone https://github.com/mikeydevx/EcoLazo.git
   ```

2. Abre la carpeta `EcoLazo` en NetBeans.
3. Antes de empezar a trabajar, actualiza tu copia local:

   ```bash
   git pull
   ```

4. Crea una rama para tus cambios:

   ```bash
   git checkout -b nombre-de-tu-rama
   ```

5. Guarda tus cambios y súbelos:

   ```bash
   git add .
   git commit -m "Describe tus cambios"
   git push -u origin nombre-de-tu-rama
   ```

6. En GitHub, abre un Pull Request hacia `main` para revisar e integrar el trabajo.
